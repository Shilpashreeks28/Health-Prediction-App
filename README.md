Health Prediction Application

AI-Powered Patient Health Analysis System built with Python, Streamlit, MySQL, and Groq AI.

1. Features
   
* Add, View, Update, Delete patient records
* AI-generated health remarks using Groq API
* Blood test analysis (Glucose, Haemoglobin, Cholesterol)
* Risk level detection (High, Medium, Low)
* Interactive charts and dashboard

2. Tech Stack
   
* Frontend & Backend: Python + Streamlit
* Database: MySQL
* AI API: Groq (LLaMA 3.3)
* Charts: Plotly

3. Setup
   
1. Install dependencies: pip install -r requirements.txt
2. Create .env file with your credentials
3. Run: python -m streamlit run app.py
